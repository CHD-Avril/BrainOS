## 2. 答辩主链路验证

浏览器验收通过以下真实项目链路：

1. 管理员登录并创建知识库。
2. 上传 `fixtures/employee-handbook.md`。
3. 后端完成 Tika 解析、切片和 1024 维向量生成。
4. 文档状态从处理中变为“可用”，切片写入真实 Chroma 容器。
5. 在 AI 问答中选择该知识库并提问年假制度。
6. 页面流式展示“5 天带薪年假”，并可展开查看原文文件名、片段和相似度。
7. 异步 SSE 结束帧、Spring Security 异步派发与消息持久化正常完成。

端到端测试使用仅在测试进程启动的 OpenAI 兼容模型桩，确保没有外网和模型额度时也能稳定重复验证；生产运行仍通过环境变量连接千问或 DeepSeek，文档存储、数据库与 Chroma 均使用真实实现。
